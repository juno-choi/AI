[codex](https://openai.com/ko-KR/codex/)가 앱 형태로 새롭게 출시되었습니다.  
  
### 간단한 특징  
- antigravity의 agent manager 기능과 거의 동일하게 사용 가능  
- git worktree 기능 지원  
- automations (crontab과 같은 스케줄링 기능)  

정도로 기존에 antigravity와 git worktree를 사용해본 경험이 있으시다면 큰 변화가 있다고 볼순 없습니다!
아마 곧 sonnet5가 출시될 예정인데 그것에 대비하여 선 출시한 것이 아닐까 싶습니다.
또한 gpt가 범용적인 ai -> 개발자용 초점 강화 흐름으로 보입니다.

  
  
### 우리가 알고 가면 좋을 점  
- antigravity의 agent manager 부터 나오던 얘기인데 이제는 코드보다는 프롬프트가 더 중요해지는 흐름입니다.  
- codex도 프롬프트 위주이고 코드는 필요할때 볼 수 있도록 기능으로 추가되어 있습니다.  
- moltbot에서 처음으로 구현된 스케줄링 기능이 ai에서 제공하는 기본 기능으로 점점 확대되고 있는 것 같습니다.  
  
### 앞으로의 흐름(개인적인 생각)  
- ai를 활용하는 개발자일 수록 더 중요해지는 스킬이 생겨나는 중  
- 여러개의 프로젝트를 병렬로 처리하는 능력 뿐만 아니라 git worktree를 사용하여 하나의 프로젝트를 병렬로 처리하는 능력  
- 프롬프트를 작성하여 기능을 빠르게 생산해내는 능력(코드 작성 능력보다 프롬프트를 통해 생산해내는 능력)  
- 하나의 ai만 사용하는 것이 아닌 여러 ai와 tool을 조합하여 최대의 생산력을 끌어내는 능력
